*Нужно взять эту сетку блоков, найти самый большой участок из смежных блоков одного цвета и вычислить, из скольких блоков он состоит.*

```
gre gre blu gre gre
gre blu blu red gre
blu gre gre red blu
blu red blu gre blu
blu red gre blu red

001 001 002 003 003
001 002 002 005 003
006 007 007 005 008
006 009 010 011 008
006 009 012 013 014

blu(002) = 3
```
